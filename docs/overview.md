# Project Overview

## Goal

Build a realistic small business network in Packet Tracer with separate departments, wireless access, DHCP, NAT, DNS, and basic security controls.

## Final topology

- HR on VLAN 10
- IT on VLAN 20
- Sales on VLAN 30
- Employee WiFi on VLAN 40
- Guest WiFi on VLAN 50
- Internal DHCP server on the business side
- ISP-side DNS and web server
- Main router connected to a simulated ISP router

## Key learning outcomes

- VLAN segmentation
- Router-on-a-stick
- DHCP relay with helper addresses
- ACL-based access control
- NAT overload
- DNS and HTTP service setup
- Wireless segmentation
