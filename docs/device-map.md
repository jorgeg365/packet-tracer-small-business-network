# Device Map

## Main devices

- Main router: 2911
- Core switch: 2960
- Internal server: DHCP and HTTP for the business side if needed
- Employee access point: VLAN 40
- Guest access point: VLAN 50
- ISP router: 2911
- ISP server: DNS and web hosting

## Switch ports used

- Fa0/1: trunk to main router
- Fa0/2: HR PC
- Fa0/3: IT PC
- Fa0/4: Sales PC
- Fa0/5: internal server
- Fa0/6: employee AP
- Fa0/7: guest AP
